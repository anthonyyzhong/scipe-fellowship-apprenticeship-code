# scipe-fellowship-apprenticeship-code
Hawaiʻi Apprenticeship Supply & Demand Analysis

Does Hawaiʻi's registered apprenticeship pipeline produce workers in the trades the state's own labor forecasts say it actually needs? This project cleans, merges, and analyzes two federal/state datasets to find out — and spends as much effort on where the data can mislead you as on the headline numbers themselves
